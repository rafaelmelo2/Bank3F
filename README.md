# BANK 3F

Projeto realizado para trabalho final da matéria de Programação orientada a objetos

### 📋 Pré-requisitos

- ENTRAR NA PASTA SRC/CONNECTOR E INSTALAR TODOS .JAR
- JDK 20.
- CRIAR BANCO DE DADOS LOCAL

execute dist/BancoPOO.jar
